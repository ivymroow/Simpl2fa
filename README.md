WHAT WAS USED AND HOW SIMPL2FA WAS MADE:
TECH STACK: BUILT WITH ELECTRON (CHROMIUM + NODE.JS) FOR CROSS-PLATFORM DESKTOP SUPPORT. THE UI USES VANILLA HTML/CSS/JS WITH THE WEB CRYPTO API FOR TOTP CODE GENERATION AND AES-256-GCM FOR VAULT ENCRYPTION.
ARCHITECTURE: TWO-PROCESS MODEL — A MAIN PROCESS (NODE.JS) HANDLES FILE I/O, ENCRYPTION, CLIPBOARD, AND SYSTEM DIALOGS. THE RENDERER PROCESS RUNS THE UI WITH FULL CONTEXTISOLATION FOR SECURITY. COMMUNICATION GOES THROUGH A TIGHTLY CONTROLLED PRELOAD BRIDGE.
SECURITY: YOUR VAULT IS ENCRYPTED WITH AES-256-GCM USING A KEY DERIVED FROM YOUR MASTER PASSWORD VIA PBKDF2 (250,000 ITERATIONS). SECRETS ONLY EXIST IN MEMORY WHILE UNLOCKED. WHEN YOU LOCK THE APP, EVERYTHING IS RE-ENCRYPTED AND WRITTEN TO DISK—NO PLAINTEXT TOUCHES STORAGE.


FEATURES:
TOTP 2FA CODES - GENERATE TIME-BASED ONE-TIME PASSWORDS USING HMAC-
SHA1/SHA256/SHA512. SUPPORTS 6 AND 8 DIGIT CODES WITH CUSTOM PERIODS. LIVE COUNTDOWN TIMER ON EVERY CODE. RIGHT-CLICK TO COPY ANY CODE INSTANTLY.
ACCOUNT SEARCH - FILTER YOUR ACCOUNTS BY ISSUER OR USERNAME WITH A LIVE SEARCH BAR AT THE TOP OF THE CODES TAB. FINDS AS YOU TYPE.
PASSWORD GENERATOR - CREATE STRONG RANDOM PASSWORDS WITH CONFIGURABLE LENGTH (8-64),
CHARACTER SETS, AND AMBIGUOUS CHARACTER EXCLUSION. BUILT-IN STRENGTH METER AND ONE-CLICK COPY.
SECURE NOTES - STORE SENSITIVE NOTES ENCRYPTED ALONGSIDE YOUR 2FA DATA. FULL CRUD WITH THE SAME AES-256-GCM PROTECTION AS YOUR VAULT.
QR CODE IMPORT - SCAN 2FA QR CODES FROM IMAGE FILES (PNG, JPEG). JUST PICK THE IMAGE AND THE APP DECODES IT USING JSQR TO EXTRACT THE ACCOUNT.
IMPORT/EXPORT-IMPORT ACCOUNTS FROM PLAIN TEXT (OTPAUTH:// URIS), JSON FILES, OR ENCRYPTED .S2FA VAULTS. EXPORT AS ENCRYPTED BACKUP OR PLAIN TEXT FOR MIGRATION TO OTHER APPS.
AUTO-LOCK — VAULT AUTOMATICALLY LOCKS AFTER A CONFIGURABLE PERIOD OF INACTIVITY (1 MIN TO 1 HOUR). TIMER RESETS ON ANY MOUSE OR KEYBOARD ACTIVITY.
MASTER PASSWORD - SINGLE PASSWORD UNLOCKS YOUR ENTIRE VAULT. CHANGE IT ANYTIME FROM SETTINGS. LOCK THE APP WHEN STEPPING AWAY.
BACKUP REMINDER - SHOWS THE LAST EXPORT DATE IN SETTINGS SO YOU ALWAYS KNOW WHEN YOUR VAULT WAS LAST BACKED UP.
